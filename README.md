# Lucky-Mart Shopping Management System 🛒

Welcome to my first C++ project! This is a console-based application designed to model a real-world shopping scenario using **Object-Oriented Programming (OOP)**.

## 📌 Project Overview
The **Lucky-Mart Shopping Management System** provides a comprehensive interface for browsing products, managing a shopping cart, and calculating bills. [cite_start]It also includes a promotional "Lucky Draw" feature. [cite: 10, 12]

## 🛠️ Key Technical Features
* [cite_start]**OOP Pillars:** I have implemented Inheritance, Encapsulation, Polymorphism, and Abstraction to make the code modular and scalable. [cite: 8, 21]
* [cite_start]**Dynamic Cart Management:** The system uses a **Linked List** to manage the shopping cart dynamically. [cite: 17]
* [cite_start]**Hierarchical Catalog:** Products are categorized into Electronics, Clothing, and Grocery. [cite: 16]
* [cite_start]**Polymorphic Display:** A single interface call correctly displays unique details for different product types. [cite: 19]
